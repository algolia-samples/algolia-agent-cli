# Contributing

Thanks for your interest in contributing to `algolia-agent`!

## Getting started

```bash
git clone https://github.com/algolia-samples/algolia-agent-cli
cd algolia-agent-cli
pip install -e ".[dev]"
```

## Running tests

```bash
pytest tests/ -v
```

## Submitting changes

1. Fork the repo and create a branch from `main`
2. Make your changes and add tests where appropriate
3. Ensure `pytest tests/` passes
4. Open a pull request with a clear description of what changed and why

## Reporting issues

Please use [GitHub Issues](https://github.com/algolia-samples/algolia-agent-cli/issues).
Include the command you ran, the output, and your Python version (`python --version`).
